this is a mini lms system. You can choose as a teacher or student. 

As a teacher, you can:
1. Create quiz
2. Start and end quiz
3. Check the stat of each quiz

As a student, you need to wait for quiz to open and do it

The backend is run on MongoDB Altas. fill in the .env file with your own `MONGO_URL`

1. Run the backend

`cd lms-backend`

`npm install`

`npm run dev`

2. Run the frontend

`cd lms-frontend`

`npm install`

`npm run dev`
